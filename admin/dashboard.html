How to Add Posts — Step by Step

For List Items (Quick links on homepage/section pages):

1. Use the generator above to create the HTML code
2. Open the correct HTML file (index.html or jobs.html etc.) in Notepad
3. Find the comment: NEW JOBS GO HERE — PASTE AT TOP
4. Paste the generated code AFTER the comment
5. Save → Upload to GitHub → Live in 30 seconds

For Detailed Posts (Full posts with tables in all_govt_job_posts.html):

1. Ask Claude: "Write a detailed HTML post for [exam name] in my SarkariAlert.in style"
2. Claude writes the complete HTML
3. Open all_govt_job_posts.html in Notepad
4. Find: PASTE NEW DETAILED POSTS BELOW THIS LINE
5. Paste Claude's HTML there
6. Also add a line to the TOC at the top
7. Save → Upload to GitHub → Done
